================================================================================
ESWBS — EXPANDED SHIP WORK BREAKDOWN STRUCTURE CODE LIST
================================================================================

Source: S9040-AA-IDX-010/SWBS 5D (Navy standard)
Converted from ESWBS_potential_codes.xls for reference.
Total codes: ~1,960 across 9 ship system groups + capability taxonomy.

Structure:
  4-digit codes (1000-9900) = Combat capability / functional taxonomy
  5-digit codes (1xxxx-9xxxx) = Ship system codes by SWBS group:
    1xxxx = Group 1: Hull Structure
    2xxxx = Group 2: Propulsion Plant
    3xxxx = Group 3: Electric Plant
    4xxxx = Group 4: Command & Surveillance
    5xxxx = Group 5: Auxiliary Systems
    6xxxx = Group 6: Outfit & Furnishings
    7xxxx = Group 7: Armament
    8xxxx = Group 8: Integration / Engineering
    9xxxx = Group 9: Ship Assembly & Support Services
  Codes with 'X' suffix = per-shaft notation (X = shaft number)

Remarks are included inline in [brackets] where present.
