================================================================================
HOW THE ALTERATION SYSTEM WORKS
================================================================================

The Navy uses a layered numbering system for ship modifications:

  SA  = Ship Alteration       — Physical modifications to ship systems.
                                Numbered 5-digit codes with optional suffix
                                (e.g., SA-94979K). The "K" suffix typically
                                denotes a kit version. Each SA maps to a
                                specific system change on a specific ship class.

  MA  = Machinery Alteration  — Modifications specific to machinery/HM&E
                                systems. Numbered 3-5 digits (e.g., MA-513).
                                Subset of SAs focused on propulsion, electrical,
                                and auxiliary machinery.

  SCD = Ship Change Document  — The engineering/design document that defines
                                the alteration. Each SA/MA has a corresponding
                                SCD. Sometimes the SCD number maps directly
                                (MA-513 = SCD-5013, SA-94979K = SCD-24979).

  SWD = Software Designator   — Software changes that accompany hardware
                                alterations (e.g., SWD-93908 paired with
                                SA-86937 IBS tech refresh).

  ORDALT = Ordnance Alteration — Modifications to weapons/ordnance systems
                                 (e.g., ORDALT-83135 for CIWS cable upgrade).

These codes are assigned by NAVSEA and tracked in the Ship Alteration
Record (SAR) database. The codes themselves are NOT in FPDS as structured
fields — they appear in free-text contract descriptions.

The DDGM BF (DDG Modernization Backfit) program bundles multiple SAs into
"core alteration" packages installed per ship during availabilities, typically
5-10 alterations per hull per cycle.

================================================================================
LIMITATIONS
================================================================================

1. This list is INCOMPLETE. FPDS descriptions are truncated at ~250 characters.
   Many contracts list 5-10 alterations but only the first few are visible.
   The DDG-90 (Chafee) contract lists "SA 71615K SA 71726K SA 77427K SA 94979K
   AND SA 84226K" — five codes visible. Other contracts installing 10+
   alterations show only the first 2-3 before truncation.

2. The SA/MA numbering system is a Navy-internal catalog. The full list of
   DDG-51 class alterations (likely hundreds) is maintained in NAVSEA's Ship
   Alteration Record (SAR) database, which is not public.

3. What each code DOES is only partially visible from contract descriptions.
   SA-86937 is identifiable as "IBS Technology Refresh" because the description
   says so. SA-71615K is only identifiable as "DDGM BF core alteration" — the
   specific system it modifies would require the SAR database or the SCD
   engineering document.

4. The SCD-to-SA mapping is not always visible. We can link MA-513 to SCD-5013
   and SA-94979K to SCD-24979 because they appear together in descriptions.
   Other mappings would require Navy planning documents.

5. Alteration codes for combat system upgrades (Aegis baseline changes, SEWIP
   block upgrades, SPY-6 backfit) are procured on their own dedicated contract
   vehicles and typically don't use SA/MA numbering in the FPDS description —
   they use program names instead (e.g., "AEGIS MODERNIZATION MMSP-CR FOR
   DDG 82" rather than an SA code).
