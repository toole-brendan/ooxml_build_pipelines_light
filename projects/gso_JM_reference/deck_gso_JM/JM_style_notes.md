## Slide 2 — Bridge Charts (2/2): MR product-tanker annual opex comps

- Match boxes to chart bar colors where appropriate  
  <sub>`id=9`</sub>
- Include currency, size (K, M, B), and real vs. nominal – for real, include year e.g., “’25E”; for nominal, state “nominal”)  
  <sub>`id=11`</sub>
- Put data rows in circles – if shaded, add in legend up top explaining the color  
  <sub>`id=12`</sub>
- Tie notes to the page using superscript numbers; do not tie sources  
  <sub>`id=13`</sub>
- Logos should be approximately the same size  
  <sub>`id=14`</sub>
- “- -” for 0  
  <sub>`id=15`</sub>
- round down if decimal <0.5; round up if decimal >0.5  
  <sub>`id=16`</sub>
- Use 1-pt dashed lines to separate content on a page  
  <sub>`id=18`</sub>
- Upper bound ranges generally hashed or dotted, but subject to content  
  <sub>`id=19`</sub>
- If multiple segments in a bar, include segment values  
  <sub>`id=20`</sub>

## Slide 4 — Tables: opex categories driving the cost differential

- Bold headers  
  <sub>`id=8`</sub>
- Bold row titles as appropriate  
  <sub>`id=9`</sub>
- Differing font sizes acceptable to emphasize certain points and/or if certain cells have minimal info  
  <sub>`id=10`</sub>
- Bullet lists; use dots for 1st level  
  <sub>`id=11`</sub>
- Use dashes for second level  
  <sub>`id=12`</sub>
- All images should have shadow in lower right hand side; ensure images are the same size  
  <sub>`id=13`</sub>
- No note / source box acceptable if sources are included in content (useful for tables)  
  <sub>`id=14`</sub>
- Place verbatims / quotes in quotation marks  and italicize  
  <sub>`id=15`</sub>
- Denote gaps in quotes with ellipses  
  <sub>`id=16`</sub>
- Table lines:  Top border - 1 pt, solid, true black Internal borders – ½ pt, solid (for content), gray #5 from template color  (LHS color scale, at the bottom - #7F7F7F) Generally avoid vertical borders for content, but may be acceptable in certain circumstances - reference other materials for examples  
  <sub>`id=17`</sub>
